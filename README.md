🚀 8-Bit Arithmetic Logic Unit (ALU) Using Verilog HDL

📖 Project Overview

This project presents the design and functional verification of an 8-Bit Arithmetic Logic Unit (ALU) using Verilog HDL. The ALU is a fundamental component of digital processors that performs arithmetic and logical operations based on a control (opcode) signal. The design was implemented using RTL methodology and verified through simulation to ensure correct functionality. This project provided practical experience in digital design, RTL coding, simulation, debugging, and verification using industry-standard tools.

🛠️ Tools Used

- Verilog HDL – RTL Design
- Visual Studio Code – Code Editor
- Icarus Verilog – Simulation
- GTKWave – Waveform Analysis
- Xilinx Vivado – RTL Design and Project Development
- Git & GitHub – Version Control

📥 Inputs

a [7:0] – First 8-bit input
b [7:0] – Second 8-bit input

📤 Outputs

Result [7:0] – 8-bit output of the selected operation

⚙️ Operations
The ALU performs multiple arithmetic and logical operations based on the selected opcode, including:

Addition - 000
Subtraction - 001
Bitwise AND - 010
Bitwise OR - 011
Bitwise XOR - 100
Bitwise NOT - 101
Left Shift - 110
Right Shift - 111


🧪 Results

The design was successfully simulated using Icarus Verilog, and the outputs were verified using GTKWave. The simulation results confirmed that the ALU performs the selected operations correctly for different input combinations.
